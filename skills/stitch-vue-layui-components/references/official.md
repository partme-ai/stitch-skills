## Usage in This Skill

1. Map Stitch HTML to Layui-Vue components (`lay-btn`, `lay-input`, `lay-card`, `lay-page-header`, `lay-result`, etc.) per [references/contract.md](contract.md).
2. Follow Layui color tokens and radius (2px/4px); use spacing grid 10px/15px.
3. Prefer Layui-Vue component API over raw HTML for buttons, inputs, cards, tables.
