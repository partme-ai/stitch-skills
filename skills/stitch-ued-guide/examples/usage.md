# Usage Example

## 1. Consult for Visual Vocabulary

**User Input:**
> "How do I describe a layout that looks like Pinterest?"

**Agent Action:**
1.  Consult `stitch-ued-guide`.
2.  Find term: "Masonry".

**Response:**
> "In Stitch prompt engineering, that's called a **Masonry Layout**. You should use: 'Masonry grid layout with staggered card heights' in your prompt."

## 2. Consult for Prompt Strategy

**User Input:**
> "My prompts are generating messy designs. Any tips?"

**Agent Action:**
1.  Consult `stitch-ued-guide` -> "Prompt Engineering Strategy".

**Response:**
> "Try structuring your prompt using the **Context -> Layout -> Component -> Content** formula. For example, instead of mixing everything, define the global style first, then the macro layout, and finally the specific components."
