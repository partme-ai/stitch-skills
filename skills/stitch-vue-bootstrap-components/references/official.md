## Usage in This Skill

1. Map Stitch HTML to Bootstrap Vue components (`b-container`, `b-row`, `b-col`, `b-button`, `b-card`, etc.).
2. Prefer Vue 3–compatible usage ([BootstrapVue Vue 3](https://bootstrap-vue.org/vue3)); long-term consider BootstrapVueNext (Bootstrap 5 + Vue 3).
3. Use design tokens and layout from the official docs when extending [references/contract.md](contract.md).
