## Usage in This Skill

1. Map Stitch HTML to uView 2 (`u-*`) components (`u-row`, `u-col`, `u-navbar`, `u-button`, `u-form`, `u-input`, etc.) per [references/contract.md](contract.md).
2. Use uView theme variables (u-primary, u-main-color, u-border-color) and spacing/radius from the docs.
3. Register pages in `pages.json`; use rpx where appropriate for responsive layout.
